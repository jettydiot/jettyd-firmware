# Jettyd Firmware SDK

ESP32 firmware SDK for connecting devices to the Jettyd platform.

**Status:** Placeholder — implementation planned for Phase 2.

## Target hardware

- ESP32-S3 (primary)
- ESP32-C3 (low-cost)
- ESP32-C6 (WiFi 6 + Thread/Zigbee)

## Planned features

- MQTT connection to Jettyd broker
- Fleet provisioning (auto-registration)
- Telemetry publishing
- Command handling
- OTA firmware updates
